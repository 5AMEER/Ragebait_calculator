## What is this?

RageBait Calculator is a fully functional-looking calculator that **never gives you the right answer.** Every time you hit `=`, one of 8 chaos modules fires instead — wrong answers, screen collapses, fake Windows crashes, shuffling keys, and more.

Built for Gen Z meme culture. Shareable via link. Completely free to use.

---

## Features

### 🎰 The Chaos Deck
Every ragebait is drawn from a shuffled deck — no module repeats until every other one has fired first. Think of it like a card deck: fully random order, but guaranteed variety.

| Module | What happens |
|---|---|
| **Wrong answer** | Confidently returns nonsense (potato, skill issue, ratio...) |
| **Key shuffle** | All buttons scramble to random positions, then drift back |
| **On fire** | Calculator catches fire with a glow and shake |
| **Fake crash** | Full Windows BSOD with a fake progress bar |
| **Glitch** | Screen corrupts with hue-rotate + zalgo text |
| **UI collapse** | Calculator squishes to nothing, springs back |
| **Fake loader** | Stalls with "Consulting the math gods..." before trolling you |
| **Key shake** | Violent shake + wrong answer |

### 🔢 Special Numbers
Certain results always trigger easter eggs, bypassing the deck entirely:

| Number | Reaction |
|---|---|
| `80085` | 😏 |
| `69420` | 💀 certified classic |
| `67` |  ayo?? |
| `42` | The Answer. to life, the universe, and everything |
| `666` | 😈 oh no. |
| `1337` | l33t hax0r |

### 💬 Roast Toasts
Wrong answer results have a 60% chance of pairing with a roast popup:
> *"bro really pulled out a calculator for this 💀"*
> *"the audacity. the AUDACITY."*
> *"sir this is a Wendy's"*

...and 11 more.

## Tech Stack

- **Vanilla HTML/CSS/JS** — zero dependencies, zero build step
- **Single file** — the entire app is one `index.html`
- **PWA-ready** — works offline, installable on mobile
- **Web Audio API** — (coming in Phase 4)

---

## Deployment

Hosted on **Vercel** — auto-deploys on every push to `main`.

To run locally, just open `index.html` in any browser. No server needed.
